Data: 
Column 1 - numbers presented
Column 2 - was the button pressed? 0 = no, 1 = yes
Column 3 - the sample of the audio clip at which the number was presented
column 4 - if a button was pressed, the reaction time 


In *Scores.mat files
the ALLSCORES variable has four rows:
row 1: correct inhibition
row 2: correct response
row 3: incorrect inhibition
row 4: incorrect response
